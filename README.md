# NameCross
Make a cross maze to include everyone's name!
把每个人的名字交叠成一个交叉迷宫！

## Perfect Example
As far as I can see, the maze below is perfect:
我觉得下面这个迷宫是完美的：

```
------------
------------
------------
----wdlsry--
--xjxym-----
cjkphrwswn--
szlscsyjx---
lyfscyylyc--
tybtdsszc---
-tjyjzqg----
whh-cyyz----
zmh---b-----
```

Where `-` represents an unused block.
其中`-`代表一个未被使用的空格
